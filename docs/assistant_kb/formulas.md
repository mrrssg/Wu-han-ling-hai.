# 公式集

## Lowes 定价公式（铁律，全系统唯一出口）
1. 成本 = 豪雅(Costway)供应商价 × 0.75
2. 退货成本预估 = (基础退货运费 + 超大件附加) × 10%（超大件附加按尺寸/重量规则自动计算）
3. 公式价 = (成本 + 退货成本预估) ÷ 除数
   - 除数 = 1 − 佣金15% − 目标毛利：12%档→0.73、15%档→0.70、18%档→0.67
4. 折扣价 = 公式价四舍五入到整数 − 0.02（如 96 → 95.98）
5. 原价 = 折扣价 ÷ 折扣系数0.5（即原价=折扣价×2）
价格永远只按这条公式算，公式之外不加任何调整机制（不设涨跌幅限制），改价靠人工确认把关。

## Macy 定价公式差异
公式价 = (成本×0.92 + 退货成本预估) ÷ 0.6444，其余步骤相同。

## 实际毛利（飞书口径）
实际毛利 = (售价×85% − 成本) ÷ 售价 ≈ (实际到账 − 成本) ÷ 售价
名义12%档的实际毛利约16%：公式里预留的退货运费预估不是每单真花掉 + 取整时向上凑，都变成额外毛利。每个SKU的实际毛利不同（退货运费不同）。

## 退货损失率（定价用）
退货损失率 = (1−p) × 窗口退货成本货值 ÷ 窗口销售额
- 窗口 = 下单满30天的订单（30~120天前）
- p = 有跟踪号退货货值 ÷ 全部退货货值（近90天）
- 分子是成本货值（付供应商的货款）不是卖价；分母是毛销售额（含被退的单）

## 修正净利率（考核口径，运营×店铺 cell，滚动90天）
修正净利率 = (非退货毛利 − 确认退货损失 − 待追敞口×(1−回收率) − 未成熟订单预扣) ÷ 销售额
考核基线 = 10%，每个运营在每个店铺各自达标。

## 每笔退货的损失
- 有「供应商退款」回填：损失 = (成本−退款) + 退货运费
- 未回填：预计口径 = 成本×(1−回收率)+运费；实际口径 = 成本全额+运费
- Lowes退货运费=0（买家退实体店）；退回我们海外仓的（有跟踪号）不欠供应商货款，损失只有退货运费≈售价×10%
- 待追回超180天 → 核销全损

## 回收率（按 店铺×供应商）
- 司顺(Vevor)：任何店铺都不退货值，回收率恒0
- Macy×豪雅(Costway)：max(实测, 90%)，当前实测约94%
- Lowes×豪雅：按实测逐笔看，不写死

## 建议提价幅度（行动清单提价候选）
建议% = (10% − 当前净利率) ÷ 0.7（摊回佣金），1%~8%封顶。

## 月净利
月净利 = 该月非退货订单利润合计 − 该月订单的退货损失。售价=整单总价（不乘数量）。
