# 利润口径与退货判定规则（全部为老板拍板的定案）

## R1 退货怎么才算真退货
飞书订单上「预估退货运费有值 且 退货原因非空」只是**申请过退货的标记**，真退货要账单坐实：
- 标记 且 实际到账≤1 → **确认退货**（货款被扣回了）
- 标记 且 实际到账为空 → **暂按退货预判**（账单没导入，先计期望损失，账单到了自动定案）
- 标记 且 实际到账>1 → **不算退货**（state=not_charged，按正常单算利润）
**Lowes例外**：Lowes平台扣款慢，退货标记+到账>1的单，从退货日起120天内仍按退货算（扣款等待期）。

## 退货台账四种状态（return_case.state）
- pending：等供应商退款
- recovered：供应商已退款
- written_off：超180天没追回，核销全损
- not_charged：标记了退货但钱没被扣走，不算退货
- warehouse：退回了我们的海外仓（有退货跟踪号），不欠供应商货款，损失只有退货运费≈售价10%

## R2 供应商退款回收率
- **司顺(Vevor)：任何店铺都不退货值，回收率0**
- **Macy店×豪雅(Costway)：max(实测,90%)**，当前实测约94%
- **Lowes×豪雅：逐笔看「供应商退款」字段，按实测**，不写死
- 回收率必须按 (店铺,供应商) 分开算，不许混算

## R4 利润口径
- 实际数据完整用「实际利润」，残缺回退「预估利润」（残缺=实到0且供退空且实利<预利，或实际成本空）
- 售价 = 整单总价，**不乘数量**（售价本身就是整单的）
- 剔除：取消单 / BestBuy-Delphi / Walmart / 供应商为空的单

## R5 双口径账本（月度影响页每月两行）
- **预计最终赚** = 毛利 − 预计口径损失（待追退款按回收率先记上）
- **目前实际落袋** = 毛利 − 实际口径损失（供应商退款只认已到账的）
- 两行差额 = 还压在供应商那边预计能退回的钱 = 财务追款指标

## 考核
- 基线 **10%**：修正净利率（滚动90天），按 运营×店铺 逐格考核，每人各自达标不看店均
- 热力图默认滚动90天；可切"最近3个订单月"看月度cohort
- 注意：定价的目标毛利（12/15/18）是定价参数，考核的10%是净利率底线，两者是不同的数

## 已知口径盲区（问到要如实说）
- 年轻店铺（如Lowes-Autool）的"修正净利率"没有链梯预扣（历史不足120天），90天口径会被未成熟新单撑高；成熟月份的真实数以月度cohort为准
- HD三店没接退货数据，看板上HD利润率偏高
- Macy的not_charged残缺单污染未修（只修了Lowes）
